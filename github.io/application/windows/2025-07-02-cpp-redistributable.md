#### C++ Redistributable 이란?

- Visual C++ 재배포 가능 패키지
- Visual C++로 개발된 응용 프로그램이 Visual C++이 설치되지 않은 컴퓨터에서도 작동할 수 있도록 하는 표준 라이브러리

#### 안에 무엇이 들어있나?

- C런타임 (CRT) : C 언어 기반 프로그램의 기본 실행 환경
- 표준 C++ 라이브러리 : C++ 언어의 표준 기능들을 사용하는 데 필요한 라이브러리, 파일입출력/네트워크 통신 등을 위한 공통 기능이 DLL 파일로 제공, 문자열처리, 메모리 관리등 런타임 라이브러 제공
- ATL (Active Template Library) : COM(Component Object Model) 기반 응용 프로그램 개발에 사용되는 라이브러리
- MFC 클래스 라이브러리
- 병렬 컴퓨팅을 위한 라이브러리등
