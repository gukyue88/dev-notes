#### DXOMARK

- 스마트폰, 카메라, 렌즈 등을 과학적으로 평가하여 순위를 매기는 상업 웹사이트
- 제조사나 소비자들에게 중요한 참고 자료가 되지만, 점수가 실제 사용자의 체감 품질을 완변하게 대변하지 않는다는 의견도 있음

#### SNR

- Singal to Noise Ratio (dB)
- 높을 수록 센서의 성능이 좋다고 평가함
- 신호가 높고, 노이즈가 낮아야 함

#### MTF

- Modulation Transfer Function
- MTF Chart
  - X : Spatial Frequency, 1mm 안에 얼마나 만흥 흑밴 선 쌍(라인 페어)가 들어있는지 의미
  - Y : 0~1 (또는 0%~100%), 1에 가까울 수록 원본 피사체의 대비가 거의 손실 없이 이미지로 잘 전달되었다는 의미
