```plaintext
// 헤더 : h1 ~ h6 태그
# ~ ###### text

// 인용 : blockquote 태그
> text or markdown

// 목록 : ul + li 태그
- item 1
- item 2

// 목록(순서) : ol + li 태그
1. item 1
2. item 2

// 코드 : pre + code 태그
` 연속 세개
+ 언어(bash, json, cs, cpp, css, diff, html, python, javascript, sql, java...)

// 수평션 : hr 태그
---

// 일반 강조 : em 태그
*text*

// 강한 강조 : strong 태그
**text**

// 삭제 텍스트 : del 태그
~~text~~

// 링크 : a 태그
[text](path)
[text][reference]

// 참조 : 위 링크의 reference 역할
[text]: path

// 이미지 : img 태그
![Alt text](path)

// 표
|제목1|제목2|제목3|
|---|---|---|
|테스트1|테스트2|테스트3|
|테스트1|테스트2|테스트3|
|테스트1|테스트2|테스트3|

// 접기펴기 목록
<details>
  <summary>여기를 클릭해서 목록을 펴보세요.</summary>

  - 항목 1
  - 항목 2
    - 서브 항목 2.1
    - 서브 항목 2.2
  - 항목 3
</details>
```
