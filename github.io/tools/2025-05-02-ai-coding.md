> 결론 : 일단, Gemini code assist + Gemini CLI 를 사용

# AI 코딩 툴은 크게 2가지 종지로 나뉨

- IDE 기반 : 코드 자동 완성, 실시간 오류검사, 디버깅 도구 등 UI 관점에서 지원이 우수함.
- CLI 기반 : 파일 시스템 조작, 라이브러리 설치, 배포 자동화, git 등 개발 전체에 걸친 복잡한 작업을 지시 가능

> 두 가지를 섞어서 사용하는 하이브리드 형태로 사용하자!

# 어떤 툴을 사용할 것인가?

- cursor vs gemini vs claude
- gemini가 무료로 제공되는 범위가 많으니, gemini를 우선 사용하고 부족한 부분이 생겼을 때 다른 툴로 넘어가자!
- gemini code assist(IDE) + gemini CLI (CLI)
