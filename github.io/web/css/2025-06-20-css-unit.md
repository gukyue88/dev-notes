> 요약 : 대부분의 경우 `rem`을 사용, 필요한 경우 다른 단위 사용

#### 자주 쓰이는 단위

- 절대단위
  - px : 운영체제의 화면의 해상도의 한 점, 상대적으로 크기가 변함
- 상대단위
  - em : 내 부모 요소의 폰트 사이즈 기준 상대 크기
  - rem(root em) : 루트 요소의 폰트 사이즈 기준 상대 크기
  - vh/vw: 뷰포트 기준 상대 크기, 1vh == 1/100 viewport의 높이, 100vh == viewport 높이 전체
  - % : 부모 요소의 크기 기준 상태 크기
  - vmin/vmax : 뷰포트의 너비와 높이 중 더 작은 값 기준 상대 크기

#### `rem` : 대부분의 요소에 사용

- 텍스트 : font-size
- 레이아웃의 간격 및 크기 : margin, padding, width, height, gap

#### `px`

- 테두리 : border
- 그림자 : box-shadow, text-shadow
- 최대/최소 크기 제한 : max-width, min-height 등
- 미디어 쿼리

#### `vw`, `vh` : 전체 화면을 채우는 레이아웃에 유용

#### `vmin`, `vmax` : 화면 방향에 관계없이 일관된 비율을 유지해야 하는 경우 사용

#### `em` : **부모 요소의 폰트 사이즈**의 비례하여 크기를 조정할 때 사용

#### `%` : **부모 요소의 크기**에 비례하여 크기를 조정할 때 사용

- 부모요소 대비 컨테이너 사이즈 조정시 : width, height
