#### 라이센스

> Icon sets를 클릭하면, 라이센스가 표시되어 있음
> 가능하면 MIT 라이센스를 사용하자

- MIT : 상업적 이용 가능, 코드 공개 의무가 없음
- Apache : 상업적 이용 가능, 코드 공개 의무가 없음, 프로그램 폴더에 license 문서 추가 필요. (문서에 추가할 내용 : https://www.apache.org/licenses/LICENSE-2.0.txt)
