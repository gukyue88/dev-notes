## SQL 명령의 종류

#### DDL (Data Definition Language)

- `CREATE` : 데이터배이스, 테이블, 뷰, 인텍스 등 생성
- `ALTER` : 데이터베이스 객체 구조 변경
- `DROP` : 데이터베이스 객체 삭제

#### DML (Data Manipulation Language)

- `SELECT` : 터이터 검색
- `INSERT` : 테이블에 새로운 데이터 삽입
- `UPDATE` : 기존 데이터 수정
- `DELTE` : 테이블에서 데이터 삭제

#### DCL (Data Control Language)

- `GRANT` : 사용자에게 특정 권한 부여
- `REVOKE` : 사용자의 특정 권한 회수

#### TCL (Transacton Control Language)

- `COMMIT` : 현재 트랜잭션에서 변경된 내용을 데이터베이스에 영구적으로 저장
- `ROLLBACK` : 현재 트랜잭션에서 변경된 내용을 취소하고, 이전 상태로 되돌림
- `SAVEPOINT` : 트랜잭션 내에서 롤백할 수 있는 지점 설정
